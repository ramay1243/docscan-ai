ПРИМЕЧАНИЕ:
-----------
Динамические страницы (статьи из БД, новости из БД, вопросы из БД) 
нужно получить отдельно из базы данных или использовать sitemap.xml:

https://docscan-ai.ru/sitemap.xml

Для получения динамических страниц:
- Статьи: https://docscan-ai.ru/articles/<slug> (где slug из таблицы articles)
- Новости: https://docscan-ai.ru/news/<slug> (где slug из таблицы full_news)
- Вопросы: https://docscan-ai.ru/questions/<id> (где id из таблицы questions)

ИНСТРУКЦИЯ ДЛЯ ЯНДЕКСА:
-----------------------
1. Перейдите в Яндекс.Вебмастер: https://webmaster.yandex.ru
2. Выберите ваш сайт
3. Раздел "Индексирование" -> "Переобход страниц"
4. Скопируйте все URL выше и вставьте в поле
5. Нажмите "Добавить" и "Отправить на переобход"

АЛЬТЕРНАТИВНЫЙ СПОСОБ:
---------------------
Можно просто отправить sitemap.xml на переобход:
1. В Яндекс.Вебмастер: "Индексирование" -> "Файлы Sitemap"
2. Добавьте: https://docscan-ai.ru/sitemap.xml
3. Яндекс автоматически переобойдет все страницы из sitemap
