# DocScan - AI анализ документов

🤖 **Умный анализ документов с помощью YandexGPT**

DocScan - это веб-сервис для автоматического анализа юридических документов, договоров и соглашений с использованием искусственного интеллекта YandexGPT.

## 🚀 Возможности

- 📄 **Анализ документов**: PDF, DOCX, TXT
- 📸 **Распознавание фото**: JPG, PNG, WEBP (платные тарифы)
- 🤖 **AI-экспертиза**: YandexGPT анализирует риски и дает рекомендации
- 👤 **Система пользователей**: Индивидуальные лимиты и статистика
- 💰 **Гибкая тарификация**: Бесплатный и платные тарифы
- 🔐 **Админ-панель**: Управление пользователями и статистикой

## 🛠 Технологии

- **Backend**: Flask, Python
- **AI**: YandexGPT, Yandex Vision API
- **Frontend**: HTML, CSS, JavaScript
- **База данных**: JSON-файлы
- **Платежи**: ЮMoney
