# DocScan AI - Описание проекта для продажи

## 🎯 О проекте

**DocScan AI** — полнофункциональный SaaS-сервис для автоматического анализа юридических документов с использованием искусственного интеллекта YandexGPT. Проект полностью готов к продакшену, развернут на продакшн-сервере и имеет работающую монетизацию.

**URL:** https://docscan-ai.ru  
**Статус:** ✅ Полностью рабочий и готовый к эксплуатации  
**Монетизация:** ✅ Работающая система тарифов и платежей  

---

## 🚀 Основной функционал

### 1. AI-анализ документов
- **Интеграция с YandexGPT** для глубокого анализа документов
- Автоматическое определение типа документа (аренда, трудовой, кредитный и т.д.)
- Многоуровневая оценка рисков (CRITICAL, HIGH, MEDIUM, LOW)
- Экспертная оценка по 4 направлениям:
  - Юридическая экспертиза
  - Финансовый анализ
  - Операционные риски
  - Стратегическая оценка
- Практические рекомендации по улучшению документа
- Поддержка форматов: PDF, DOCX, DOC, TXT, изображения (JPG, PNG)

### 2. Система регистрации и авторизации
- Полноценная регистрация с email и паролем
- Восстановление пароля через email
- Email-верификация с подтверждающими ссылками
- Система сессий с безопасными cookies
- Личный кабинет пользователя

### 3. Тарифная система и монетизация
- **Бесплатный тариф**: 1 анализ в день
- **Базовый тариф**: 490₽/мес — до 1000 анализов
- **Премиум тариф**: 399₽/мес — до 50 анализов
- **Безлимитный тариф**: 800₽/мес — до 1000 анализов
- **Интеграция с YooMoney** для автоматической обработки платежей
- Webhook для автоматической активации тарифов после оплаты
- Система отслеживания срока действия тарифов

### 4. Личный кабинет пользователя
- История всех анализов документов
- Статистика использования (всего анализов, анализов сегодня)
- Информация о текущем тарифе и сроке его действия
- Просмотр деталей предыдущих анализов
- Кнопка быстрой покупки тарифа
- Управление профилем (email, пароль)

### 5. Админ-панель (расширенная)
Мощная административная панель с разделением на секции:

#### 📊 Dashboard (Главная)
- Общая статистика по пользователям, гостям и анализам
- Статистика калькулятора штрафов
- Аналитика в реальном времени

#### 👥 Управление пользователями
- Просмотр всех зарегистрированных пользователей
- Выдача тарифов вручную
- Поиск пользователей по ID, тарифу, IP
- Создание новых пользователей
- Просмотр статистики каждого пользователя

#### 👤 Управление гостями
- Отслеживание незарегистрированных пользователей
- Статистика по IP-адресам и браузерам
- Связывание гостей с зарегистрированными пользователями
- Поиск по IP и User-Agent
- Мониторинг предложений регистрации

#### 📧 Email-рассылки
- Создание массовых рассылок по базе пользователей
- Фильтрация получателей (все, бесплатный тариф, платные тарифы, верифицированные)
- Шаблоны писем с переменными ({email}, {user_id}, {plan})
- Предпросмотр рассылки
- История отправленных рассылок
- Статистика отправки (отправлено, ошибки)

#### 📝 Управление статьями блога
- Визуальный редактор TinyMCE с полным функционалом
- Создание и редактирование статей через веб-интерфейс
- Загрузка изображений для статей
- SEO-оптимизация (мета-ключевые слова, описания)
- Категоризация статей
- Управление публикацией (черновик, опубликовано, архив)
- Счетчик просмотров статей
- Автогенерация URL (slug) для статей

### 6. Калькулятор штрафов
- Расчет неустойки по договорам
- Различные формулы расчета
- Сохранение истории расчетов
- Статистика использования

### 7. Блог и контент-маркетинг
- Динамическая система статей с отдельными страницами
- Категоризация и поиск
- SEO-оптимизированные страницы
- Система просмотров
- Навигация между статьями (предыдущая/следующая)
- Более 10 готовых статей на юридические темы

### 8. Система лимитов
- **IP-лимиты** для незарегистрированных пользователей (1 анализ с IP в день)
- Дневные лимиты для зарегистрированных пользователей
- Автоматический сброс лимитов через 24 часа
- Отслеживание использования по IP и User-Agent
- Система предложения регистрации после исчерпания лимита

### 9. Telegram-интеграция
- Автоматическая отправка заявок с формы обратной связи в Telegram-бота
- Настраиваемый бот для техподдержки

### 10. Email-сервис
- Отправка верификационных писем
- Восстановление пароля
- Массовые рассылки по базе
- Персонализация писем

## 🛠 Технический стек

### Backend
- **Flask 2.3.3** — веб-фреймворк
- **SQLAlchemy 3.0.5** — ORM для работы с БД
- **SQLite** — база данных (легко мигрируется на PostgreSQL/MySQL)
- **Gunicorn** — WSGI-сервер для продакшна
- **Python 3.x**

### Frontend
- Чистый HTML/CSS/JavaScript
- Адаптивный дизайн (mobile-first)
- Современный UI с градиентами и анимациями
- TinyMCE для визуального редактирования статей

### Интеграции
- **YandexGPT API** — анализ документов с ИИ
- **YooMoney API** — обработка платежей
- **Telegram Bot API** — уведомления
- **SMTP** — отправка email (Gmail, любой SMTP)

### Безопасность
- Хеширование паролей (Werkzeug)
- Сессии с безопасными cookies
- CSRF-защита
- IP-лимиты для защиты от злоупотреблений
- Валидация входных данных
