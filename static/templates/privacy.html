Политика конфиденциальности

Дата последнего обновления: 24 ноября 2025 года

1. Общие положения

1.1. Настоящая Политика конфиденциальности определяет порядок обработки и защиты информации о Пользователях сервиса DocScan.

1.2. Цель политики — обеспечение защиты прав Пользователей при использовании сервиса анализа документов.

2. Собираемая информация

2.1. Техническая информация:

  • Анонимный идентификатор пользователя
  • Статистика использования сервиса (количество анализов, даты использования)
  • Информация о браузере и операционной системе
  • IP-адрес (в обезличенном виде)

2.2. Обработка документов:

  • Загружаемые документы передаются в API для анализа
  • Документы не сохраняются на наших серверах после обработки
  • Текст документов удаляется сразу после формирования отчета
  • Временные файлы автоматически удаляются в течение 1 часа

3. Использование информации

3.1. Собираемая информация используется исключительно для:

  • Предоставления услуг анализа документов
  • Улучшения работы сервиса
  • Учета лимитов использования
  • Технической поддержки Пользователей

4. Защита информации

4.1. Администрация принимает необходимые организационные и технические меры для защиты информации Пользователей.

4.2. Передача данных между Пользователем и сервисом осуществляется по защищенному протоколу HTTPS.

4.3. Доступ к статистической информации имеют только уполномоченные сотрудники Администрации.

Гарантии безопасности:

  • SSL/TLS шифрование всех передаваемых данных
  • Регулярное обновление систем безопасности
  • Мониторинг несанкционированного доступа
  • Регулярное резервное копирование данных

5. Cookies и аналитика

5.1. Сервис использует cookies для:

  • Сохранения идентификатора пользователя между сессиями
  • Учета дневных лимитов использования
  • Сбора анонимной статистики для улучшения сервиса

Важно: Вы можете отключить cookies в настройках вашего браузера, однако это может ограничить функциональность сервиса.

6. Передача данных третьим лицам

6.1. Текст документов передается в API для обработки через искусственный интеллект.

6.2. Передача осуществляется в соответствии с политикой конфиденциальности.

6.3. Никакая другая информация о Пользователях не передается третьим лицам.

Обратите внимание: При использовании AI API текст ваших документов передается на серверы для анализа, но не сохраняется там постоянно.

7. Права Пользователей

7.1. Пользователь вправе запросить информацию о хранящихся о нем данных.

7.2. Пользователь может отказаться от сбора статистической информации.

7.3. Для реализации своих прав Пользователь может обратиться в техническую поддержку.
