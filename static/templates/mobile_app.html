Как установить приложение

1

Скачайте APK

Нажмите кнопку "Скачать APK файл" выше. Файл сохранится в папке "Загрузки".

2

Разрешите установку

Перейдите в Настройки → Безопасность → Неизвестные источники → Разрешить для браузера.

3

Откройте файл

Найдите скачанный файл в "Загрузках" и нажмите на него для установки.

4

Завершите установку

Нажмите "Установить" и дождитесь завершения. Готово!

Частые вопросы

Приложение безопасно?

Да, абсолютно безопасно. Это официальное приложение DocScan AI. Код проверяется на отсутствие вредоносного ПО. Приложение не запрашивает лишних разрешений.

Почему не в Google Play?

Приложение находится в процессе публикации в Google Play Store. Пока оно доступно для прямого скачивания с нашего сайта. Это стандартная практика для быстрого запуска.

Нужен ли интернет для работы?

Для сканирования документов интернет не требуется. Для анализа документов с помощью ИИ требуется подключение к интернету, так как обработка происходит на наших серверах.

Будут ли обновления?

Да, приложение будет автоматически обновляться при выходе новых версий. Мы постоянно добавляем новые функции и улучшаем работу ИИ.

Поддерживаются ли другие ОС?

Сейчас доступна только версия для Android. Версия для iOS (iPhone) находится в разработке и появится позже.
